Linux Backlight

The backlight is controlled via sysfs in Linux. Replace <backlight-device> with the appropriate directory name from /sys/class/backlight/.

# cd /sys/class/backlight/<backlight-device>
# cat max_brightness     # Displays the maximum brightness value
# cat brightness         # Displays the current brightness value
# echo 0 > brightness    # Sets brightness to minimum
# echo 50 > brightness   # Sets brightness to 50
# echo 100 > brightness  # Sets brightness to maximum (if 100 <= max_brightness)
